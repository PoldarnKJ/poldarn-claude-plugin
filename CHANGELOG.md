## [1.0.0] - 2025-01-06

### Added

- **Skills**
  - Code Quality Principles: Architectural guidance for SRP, modular design, type-first development, FP style
  - Deep Investigation: Systematic research methodology for outdated LLM knowledge issues
  - Fake Implementation Detection: Detection patterns for behavioral mimicry code
  - TypeScript Type Scanner: Error classification and type health analysis
  - Vue Query Patterns: Best practices for @tanstack/vue-query in Vue 3

- **Agents**
  - code-quality-review: Architectural review against five core principles
  - codex-reviewer: Deep code review with Codex MCP integration
  - deep-investigation: Exhaustive external research for library/framework issues
  - fake-implementation-detector: Scans for hollow implementations
  - typescript-type-scanner: TypeScript health reports with root cause analysis

- **Commands**
  - `/poldarn:ts-scan`: TypeScript type health scan command
